This website was made by Maxime Gros in his bedroom using HTML5, CSS3 and Javascript. This was a great way to develop front-end skills. Javascript with event handlers are used to create the shadow effect on the "Home" page and keyframes are used to create the movie slider effect in the "Adaptations" page.

Tolkien was chosen as many pictures, fonts, books and movies are available on this topic.
